Olá, eu sou o Matheus

sou formado em Análise e Desenvolvimento de Sistemas e atualmente estou estudando linguagem de programação Python,
focando no Framework Django, e com a necessidade de utilizar o frontend eu comecei a utilizar o Bootstrap que é um Framework
muito completo e responsivo.

Eu já tinha estudado sobre as Grids na época de faculdade e pra falar a verdade eu tinha recebido uma explicação bem difícil de imaginar
e colocar em prática, durante um curso rápido de Bootstrap eu consegui abstrair o que aprendi de uma forma visual, e explicada em HTML,
decidi então fazer esse projeto simples com o intuito de repassar esse conhecimento que é bem útil na parte visual de sites, e aproveitar
para trabalhar meu portifólio.


Este projeto foi escrito utilizando HTML e o Bootstrap, espero que ajude!

Aproveite e abra o código fonte da página para ver o detalhamento.
